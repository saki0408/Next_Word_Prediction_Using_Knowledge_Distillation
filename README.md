# Next_Word_Prediction_Using_Knowledge_Distillation
"Next Word Prediction using Knowledge Distillation with 
Transformer-teacher and LSTM-student," aimed at enhancing next-word prediction for 
resource-constrained devices. By transferring knowledge from a high-capacity Transformer to a 
lightweight LSTM, we achieved a validation accuracy of 9.0% compared to a baseline LSTM’s 
peak of 8.0%. Leveraging the "ArticlesMarch2018.csv" dataset, innovative preprocessing, and 
insights from ULMFIT, we address the challenge of efficient NLP deployment.
